# 🎯 Number Guessing Game

A Python-based command-line game where the player attempts to guess a randomly generated number between 1 and 100. The game provides hints after each attempt, helping the player reach the correct answer efficiently.

## 📌 Project Overview

This project demonstrates fundamental Python programming concepts including:

- Variables and Data Types
- Conditional Statements
- Loops
- User Input Handling
- Random Number Generation
- Basic Game Logic

## 🚀 Features

✅ Random number generation between 1 and 100

✅ Interactive command-line interface

✅ Higher/Lower hints after each guess

✅ Quit option to exit the game anytime

✅ Beginner-friendly and lightweight

## 🛠️ Technologies Used

- Python 3.x
- Random Module
